# Product Search

This is a technical test that I found online and decided to undertake to enhance my skills. The test involves developing a product search engine by consuming the 'Mercado Libre' site's API to retrieve products from its catalog and display the details of the selected product.

You can check the result deployed here:

https://pacomorando.github.io/practica-product-search


This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.0.0.
